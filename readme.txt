== Description ==
This plugin is used to specify the IP written comments, to reduce comment spam
issues, can be used for SEO, managing multiple sites.Please commments.php added before "do_action ('comment_form_before')".
== Installation ==
* PHP >= 5.2 needed.
* Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
